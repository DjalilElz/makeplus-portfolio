## ğŸš€ Quick Start

### Prerequisites
- Node.js (v18 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/DjalilElz/makeplus-portfolio.git
cd makeplus-portfolio

# Install dependencies
npm install

# Start the development server
npm start
```

Visit `http://localhost:3000` in your browser.

## ğŸ“§ Contact Form Setup

The contact form requires backend API setup. See [BACKEND_SPECIFICATION.md](BACKEND_SPECIFICATION.md) for complete implementation details.

### Quick Setup Options:

1. **Full Backend** (Recommended for production)
   - Build backend API with Express.js + MongoDB
   - Follow `BACKEND_SPECIFICATION.md`
   - Deploy to Railway or Render

2. **Quick Solution** (For testing)
   - Use Formspree or EmailJS
   - No backend development needed
   - Limited to form submissions only
