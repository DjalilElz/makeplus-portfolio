# Video Assets Folder

Place your optimized videos and poster images here.

## Required Files:

### Videos (MP4 format, H.264 codec):
- video1.mp4
- video2.mp4
- video3.mp4
(Add more as needed)

### Poster Images (JPG format):
- poster1.jpg
- poster2.jpg
- poster3.jpg
(Add more as needed)

## Quick Start:

1. **Compress your videos** using the methods in VIDEO_OPTIMIZATION_GUIDE.md
2. **Generate poster images** (thumbnails) from your videos
3. **Place files** in this directory
4. **Update** the video list in `/public/js/main.js`

## File Naming Convention:

- Videos: `video1.mp4`, `video2.mp4`, etc.
- Posters: `poster1.jpg`, `poster2.jpg`, etc.
- Keep names simple and sequential

## Recommended Specs:

- **Video Format**: MP4 (H.264)
- **Resolution**: 1920x1080 or 1280x720
- **File Size**: 20-50 MB per video
- **Poster Size**: 400x225 pixels (16:9 ratio)
